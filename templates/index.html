SP-API Reports API Playground - Beta
Introduction

This is a simple application to help you test the SP-API. The application does not save any credentials on the server.

(Optional) Install the JSON Formatter Browser Extension

With this extension, you can print JSON and JSONP nicely when you visit it 'directly' in a browser tab.
